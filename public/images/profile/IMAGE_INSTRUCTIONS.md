# Profile Photo Instructions

## Profile Photo Location

Your profile photo should be saved as:
- **File name**: `saikat-ghosh.jpg` or `saikat-ghosh.png`
- **Location**: `public/images/profile/saikat-ghosh.jpg`

## Image Requirements

- **Format**: JPG or PNG
- **Size**: Minimum 400x400px, recommended 800x800px
- **Aspect Ratio**: Square (1:1) - will be displayed as circular
- **File Size**: Keep under 500KB for best performance

## How to Add Your Photo

1. Save your profile photo to: `public/images/profile/saikat-ghosh.jpg`
2. The image will automatically appear in the Hero section
3. If the image doesn't load, check:
   - File name matches exactly: `saikat-ghosh.jpg`
   - File is in the correct directory: `public/images/profile/`
   - File format is JPG or PNG

## Current Status

The portfolio is configured to use your profile photo. Once you add the image file, it will display automatically in the Hero section with a circular border and animation.
